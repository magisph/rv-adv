# Markdown Guide

## Headers

# H1
## H2
### H3

## Emphasis

*italic* or _italic_
**bold** or __bold__
~~strikethrough~~

## Lists

- Item 1
- Item 2
  - Nested item

1. First
2. Second
3. Third

## Links

[Link text](https://example.com)

## Code

Inline `code` with backticks.
```
Code block
with multiple lines
```

> This is a quote.
