# Rule: Padrão de Commits

## Formato: Conventional Commits (em Português)

feat: adiciona módulo de controle de férias dos servidores
fix: corrige cálculo de prazo em finais de semana estendidos
docs: atualiza documentação da API de jurisprudência
style: ajusta espaçamento do formulário de benefícios
refactor: extrai lógica de cálculo para hook dedicado
perf: otimiza query de listagem de processos com índice
test: adiciona teste de validação de webhook HMAC
chore: atualiza dependências do Supabase CLI
ci: adiciona workflow de deploy automático para edge functions
security: adiciona rate limiting na função de exportação

## Branches
- feature/<nome-curto> — novas funcionalidades
- fix/<nome-curto> — correções de bugs
- hotfix/<nome-curto> — correções urgentes em produção
- refactor/<nome-curto> — refatorações
- docs/<nome-curto> — documentação
