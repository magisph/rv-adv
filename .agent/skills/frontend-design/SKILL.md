# Frontend Design & UI/UX
1. **Acessibilidade (a11y):** Garanta que botões e modais tenham contraste adequado e navegação amigável por teclado.
2. **Consistência:** Utilize o Tailwind CSS para manter a consistência de cores da paleta definida em `index.css`.
3. **Componentização:** Priorize o reuso de componentes da pasta `@/components/ui/` em vez de criar HTML cru.
